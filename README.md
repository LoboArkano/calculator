## Getting Started

To get a local copy up and running follow these simple example steps.
```
    git clone git@github.com:LoboArkano/Weather-App.git
```

### Prerequisites

- npm

- Nodejs

### Setup

Install dependencies with:

```
    npm install
```

Start app with:

```
   npm start
```

### Usage

Use the app like any calculator.
